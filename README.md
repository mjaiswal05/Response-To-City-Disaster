# Disaster Response Dashboard

A scalable emergency operations dashboard built for managing incidents, evacuations, alerts, and city workflows.

## Features
- Incident tracking
- Evacuation workflows
- Department dashboards
- Real-time notifications
- Role-based operations

## Tech Stack
React, TypeScript, Vite, Tailwind CSS, Docker, Kubernetes

## My Contribution
Developed and enhanced dashboard modules, UI components, and production-grade workflows.

## Run Locally
npm install
npm run dev
